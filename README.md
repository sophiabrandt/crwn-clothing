## Installation And Usage

Clone the repo and install dependencies:

```sh
sh -c '(pnpm install && cd client && pnpm install)'
# or:  sh -c '(npm install && cd client && npm install)'
```

Run the development server:

```sh
pnpm run dev
# or: npm run dev
```
